The files in this contain the bmconfig file for the different heating profiles, the profiles being:
301,354,411,501

These heating profiles were independently taken and Bmeconfig files were exported for the same.

Algorithm consists of Classes of data of each fruit/vegrtable taken in atleast 3 states(Good, ok Bad) and air. This way the algorithm can identify a fruit/vegetable irrespective of which state it is in.

The bsec config files were created and are stored along with the BME config file for each heating profile.
